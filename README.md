# Bastion

Manage all of your home's devices in one place.

## Getting Started
-  flutter run -d chrome --web-port=8081


## Tabs

### Dashboard
- View list of devices
- Tap to toggle on/off

### Devices
- View list of devices
- Add new device
- Edit device
- Delete device

### Routines
- View list of routines

### Event Log
- View historical list of adds, edits, and deletes

### Settings
- Change theme
